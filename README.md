# Итоговый проект: Разработка и развертывание ИИ-приложения в Kubernetes

## Цель проекта
Разработать и развернуть локально в Minikube контейнерное Java-приложение с интегрированной ИИ-моделью для анализа тональности текста, настроить мониторинг, балансировку нагрузки и проанализировать современные тенденции в области ИИ и контейнеризации.

## Задачи проекта
1. Установить и настроить Minikube для локального развертывания Kubernetes
2. Разработать REST API на Java Spring Boot для анализа тональности текста
3. Создать Docker-образ приложения
4. Развернуть приложение в Kubernetes с 3 репликами
5. Настроить LoadBalancer Service и Ingress для маршрутизации трафика
6. Реализовать Horizontal Pod Autoscaler для автомасштабирования
7. Проанализировать современные тенденции в области ИИ и контейнеризации

## Используемые технологии
- **Java 21 + Spring Boot 3.2.0** - разработка приложения
- **Docker** - контейнеризация приложения
- **Minikube** - локальный Kubernetes кластер
- **Kubernetes** - оркестрация контейнеров
- **NGINX Ingress Controller** - маршрутизация HTTP трафика
- **Prometheus + Grafana** - мониторинг и визуализация метрик
